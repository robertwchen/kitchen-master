# Experiment 001: Synthetic Geometry

## Goal

Establish a minimal, reproducible scaffold for synthetic geometry experiments that validate label logic and evaluation pipelines before real-data baselines.

## What This Scaffold Does

- Loads experiment configuration from `config.yaml`
- Creates timestamped output directories under `results/`
- Logs placeholder execution steps
- Writes an empty-structure metrics file (no fabricated values)

## What It Does Not Do

- Generate synthetic datasets
- Train or run detection models
- Produce performance claims
